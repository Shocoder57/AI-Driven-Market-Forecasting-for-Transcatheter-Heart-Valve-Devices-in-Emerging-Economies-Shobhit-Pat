# AI-Driven Market Forecasting for Transcatheter Heart Valve Devices

## Project Overview
This project develops an AI-based market forecasting framework to predict the growth potential of Transcatheter Heart Valve (THV) devices in emerging economies.

## Countries Analyzed
- India
- China
- Brazil
- South Africa

## Methodology
- Data sourced from World Bank indicators
- Composite Education Index and Market Growth Index constructed
- Machine Learning models applied:
  - Linear Regression (baseline)
  - Random Forest Regression (AI model)
